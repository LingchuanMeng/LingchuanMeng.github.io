1. 下载安装NodeJs和Hexo（基于windows）

1.1 下载NodeJs

点击这里在官网下载NodeJS安装包，网站会根据你的系统和自动选择正确的版本

  • 建议优先选择LTS+x64版本（应该没有老电脑还需要x86或arm64吧）
  • 如果下载速度很慢，建议选择去清华大学的镜像站下载（点击这里）
    • 先选择需要的版本（可以选择上面官网默认选择一样的版本，先选择版本，点进去，在选择现在哪个版本。
    • windows系统建议选择 node-v版本号-x64.msi，下载后点击即可安装

1.2 安装NodeJs及其注意事项

  • 在安装过程中可能有一个选项，是说是否允许nodeJs自动下载依赖的软件库（chocolatey），如果你真的不太懂，可以选择一下（但是可能存在一点小小的问题）参考链接
    • 如果点选了这个按钮，那么在安装完成后会自动弹出一个蓝色或黑色的窗口（powershell或命令提示行），然后程序就会自动开始下载安装进程
    • 可能存在的问题：如果你的电脑里已经有python了，这个程序可能无法检测出你电脑中的python（或者本人电脑python版本过低【3.9.6应该也不太低吧】），软件可能会自动为你下载另外一个版本的python（狗屎），这可能会导致python版本错乱（默认python版本被换成新版本，新版本python缺乏旧版本python的库，导致之前的软件爆炸）
      • 这个问题出现的原因可能时因为nodejs需要安装 chocolatey这个插件，在安装 chocolatey时会在 chocolatey的根目录下安装一个新版本python，同时将这个python的路径写入全局环境变量（高级系统设置-环境变量-系统变量-PATH的第一条）
      • 解决办法：按照（高级系统设置-环境变量-系统变量-PATH）的顺序打开，你会发现第一第二条的后缀分别是/python和/python/Script，在这两条下面是你原来的python版本的路径（如果没有的话，退出去，找到上面那个现有用户环境变量的PATH值，复制到系统变量中），然后把有 chocolatey和 python的条目下移到只有 python的条目的下方

6. 如何插入路径、链接和图片

参考链接：Hexo官网-资源文件夹

  • 打开文章资源文件夹

    • 对于那些想要更有规律地提供图片和其他资源以及想要将他们的资源分布在各个文章上的人来说，Hexo也提供了更组织化的方式来管理资源。这个稍微有些复杂但是管理资源非常方便的功能可以通过将 config.yml文件中的 post_asset_folder选项设为 true 来打开。

  • 当你打开文章资源文件夹功能后，你把一个 example.jpg图片放在了你的资源文件夹中，如果通过使用相对路径的常规 markdown 语法 ![](example.jpg)，它将 不会 出现在首页上。（但是它会在文章中按你期待的方式工作）
